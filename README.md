## Project Title
Video Game Sales Data Analysis

## Description
This project analyses a video game sales dataset using Python. The program reads a CSV file, processes the data using Pandas, displays useful statistics, sorts games by global sales, and creates visualisations using Matplotlib.

## Dataset
Dataset: Video Game Sales  
Source: Kaggle  
File used: dataset/vgsales.csv

The dataset contains video game sales data including name, platform, year, genre, publisher, and regional/global sales.

## Requirements
- Python 3
- pandas
- matplotlib

Install libraries:
```bash
python -m pip install pandas matplotlib
```
## How to Run

```bash
python src/main.py
```

## Outputs

- First 5 rows of the dataset
- Top 10 best-selling games
- Number of games
- Summary statistics
- Publisher sales graph
- Genre sales graph
